🛡 三级门控机制
Agent 校验规则 PASS 条件 RETRY 触发 STOP 触发
策略师 lift 合理性 + 字段完整 + 策略不重复 expected_lift ≤ 50% lift > 50% 疑似幻觉 重试 ≥ 2 次
执行官 埋点数量 + 样本量 + 必需字段 tracking ≥ 2 & sample ≥ 1000 埋点不足 / 字段缺失 重试 ≥ 2 次
分析师 p值范围 + lift方向一致性 + 字段完整 0 ≤ p ≤ 1 & 方向一致 p值异常 / lift矛盾 重试 ≥ 2 次
复盘官 结论四选一 + 学习实质 + 下一步方向 verdict ∈ 4 个有效值 结论无效 / 字段缺失 重试 ≥ 2 次
